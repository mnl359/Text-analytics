# Text Analytics

Program that search the number of appearances of a word in a news dataset.

## Requirements

* Python 3

## Usage

### Serial code

``` bash
python3 serial.py
```
Or check if the interpreter path is correct at the beginning of the code and give it
the necessary permission:

``` bash
./serial.py
```
